# 行動規範（Code of Conduct）

本プロジェクトは [Contributor Covenant](https://www.contributor-covenant.org) v2.1 を行動規範として採用します。

## 私たちの約束

メンバー、コントリビューター、リーダーである私たちは、年齢・体格・目に見える/見えない障害・民族・性的特徴・性自認や性表現・経験の度合い・教育・社会経済的地位・国籍・容姿・人種・宗教・性的指向にかかわらず、誰もがハラスメントのないコミュニティへ参加できることを約束します。

## 行動基準

コミュニティに良い影響を与える行動の例:

- 他者への共感と思いやりを示す
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを行い、受け入れる
- 自らの過ちに責任を持ち、影響を受けた人々に謝罪し、その経験から学ぶ
- 個人ではなくコミュニティ全体にとって最善のことに目を向ける

容認されない行動の例:

- 性的な言動や、あらゆる種類の性的な誘い・関心の押し付け
- 荒らし、侮辱・中傷的なコメント、個人的・政治的な攻撃
- 公的・私的を問わないハラスメント
- 明示的な許可なく他者の私的情報（住所やメールアドレス等）を公開すること
- 職業上不適切と合理的に考えられるその他の行為

## 適用範囲

この行動規範は、コミュニティ内のすべての場（リポジトリ、Issue、Pull Request、ディスカッション等）と、個人がコミュニティを公的に代表する場に適用されます。

## 違反の報告・対応

容認されない行動を目撃した、または受けた場合は、プロジェクトメンテナ（[@canpok1](https://github.com/canpok1)）まで GitHub 経由でご連絡ください。すべての報告は速やかに、かつ公正に確認・対応されます。報告者のプライバシーと安全は尊重されます。
