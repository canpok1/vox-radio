# [C] 演出プロンプト（SE挿入位置の判定）

以下のセリフ列と使用可能なSE一覧を元に、SE（効果音）を挿入する位置を判定してください。

OP/EDジングルやBGMの配置はコードが決定するため、ここではSEの挿入位置のみを判定してください。

## セリフ列

```json
{{lines}}
```

## 使用可能なSE一覧

```json
{{se_catalog}}
```

## 出力形式

以下のJSON形式で回答してください。

```json
{
  "se_insertions": [
    {
      "after_line_index": 2,
      "se_name": "chime",
      "reason": "コーナー開始のため"
    }
  ]
}
```

## 注意事項

- SEはトピックの転換点など、効果的な場所にのみ挿入してください
- 使いすぎると煩わしくなるので、厳選して挿入してください（全体で0〜3回程度）
- `after_line_index` は0始まりのインデックスです（0 = 最初のセリフの後に挿入）
- `se_name` は使用可能なSE一覧のキー名を使用してください
- SEが不要な場合は `se_insertions` を空配列にしてください
