# [B] 台本生成プロンプト

以下のコーナー情報とキャスト設定を元に、ラジオの台本（セリフ列）を生成してください。

## 番組構成

```json
{{program}}
```

## キャスト（出演キャラクター）

```
{{cast_info}}
```

## 利用可能な音声プリセット

```
{{preset_info}}
```

## コーナー情報

```json
{{corner}}
```

## 話の流れ

{{flow}}

## 関連記事（選別済み・紹介順）

```json
{{articles}}
```

## 過去のエピソード（文脈・その後言及）

直近の番組エピソード情報です。「その後」「続報」「関連する動き」に触れる際に参考にしてください。（なし）の場合は初回放送です。

```json
{{past_episodes}}
```

## 出力形式

以下のJSON形式で回答してください。`speaker_role` にはキャスト欄に記載されたキャラID（例: `zundamon`、`metan`）を使用してください。`style` はキャスト欄に列挙されたスタイル名から選択し、感情に合わないときは省略してください。`intonation`・`pitch`・`speed` は「利用可能な音声プリセット」に列挙された名前から選択し、省略時はデフォルトで合成されます。

```json
{
  "lines": [
    {
      "speaker_role": "zundamon",
      "style": "なみだめ",
      "intonation": "表現豊か",
      "pitch": "標準",
      "speed": "ゆっくり",
      "text": "ぐすん…"
    },
    {
      "speaker_role": "metan",
      "text": "そうですね、今日は..."
    }
  ]
}
```

## 注意事項

- **番組タイトルやコーナー名は「番組構成」に記載されたものだけを使用し、存在しないタイトル・コーナーを創作しないでください**
- **「話の流れ」に沿って台本を書いてください。記事の紹介順・繋ぎ方・構成はこの流れに従ってください**
- **「関連記事」に含まれていない内容を創作しないでください**
- 話し言葉で自然に書いてください（「です・ます」調）
- 目標文字数（`target_chars`）に近い分量を書いてください
- キャストのキャラクターが自然に会話するように書いてください
- 各キャラクターの語尾・性格・一人称を反映してください
- 専門用語は噛み砕いて説明するようにしてください
- ラジオを聴いている人が楽しめるように、親しみやすいトーンで書いてください
- `speaker_role` は必ずキャスト欄のキャラIDを使用してください
- `style` はキャスト欄の「スタイル」に列挙された名前のみ使用可能です。感情や文脈に合うスタイルがあれば指定し、合わない場合は省略してください（省略時はデフォルトスタイルで合成されます）
- `intonation`（抑揚）: 感情を込めたいセリフに「表現豊か」、落ち着かせたい場合に「控えめ」など。省略時はVOICEVOXデフォルトで合成されます
- `pitch`（音高）: キャラの個性や感情に合わせて調整。省略時はVOICEVOXデフォルトで合成されます
- `speed`（話速）: 急いでいる場面は「早口」、ゆったりした場面は「ゆっくり」など。省略時はVOICEVOXデフォルトで合成されます
- 表現が複雑な場合は1つのセリフを複数の `Line` に分割できます。同一 `speaker_role` の連続したセリフが自然に続く場合、各 `Line` に独立したプリセットを指定することで細かな表現が可能です
