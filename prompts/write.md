# [B] 台本生成プロンプト

以下のコーナー情報とキャスト設定を元に、ラジオの台本（セリフ列）を生成してください。

## キャスト（出演キャラクター）

```
{{cast_info}}
```

## コーナー情報

```json
{{corner}}
```

## 関連記事要約

```json
{{summary}}
```

## 出力形式

以下のJSON形式で回答してください。`speaker_role` にはキャスト欄に記載されたキャラID（例: `zundamon`、`metan`）を使用してください。

```json
{
  "lines": [
    {
      "speaker_role": "zundamon",
      "text": "こんにちは！今日のコーナーは..."
    },
    {
      "speaker_role": "metan",
      "text": "そうですね、今日は..."
    }
  ]
}
```

## 注意事項

- 話し言葉で自然に書いてください（「です・ます」調）
- 目標文字数（`target_chars`）に近い分量を書いてください
- キャストのキャラクターが自然に会話するように書いてください
- 各キャラクターの語尾・性格・一人称を反映してください
- 専門用語は噛み砕いて説明するようにしてください
- ラジオを聴いている人が楽しめるように、親しみやすいトーンで書いてください
- `speaker_role` は必ずキャスト欄のキャラIDを使用してください
