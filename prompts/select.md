# [A] 記事選別・話の流れ設計プロンプト

以下のコーナー情報と候補記事を元に、番組で実際に扱う記事を選別し、話の流れを設計してください。

## コーナー情報

```json
{{corner}}
```

## 候補記事（タイトルと URL のみ）

```json
{{articles}}
```

## 過去に扱った記事 URL（重複排除）

以下の URL は過去の番組で既に紹介済みです。リスナーが聴き飽きないよう、原則として選別から除外してください。

```json
{{past_urls}}
```

## 出力形式

以下のJSON形式で回答してください。

```json
{
  "selected_urls": [
    "https://example.com/article/1",
    "https://example.com/article/2"
  ],
  "flow": "まず記事1でAIチップの性能向上を紹介し、次に記事2で普及の課題を取り上げて締めくくる"
}
```

- `selected_urls`: 番組で紹介する記事の URL を**紹介順**に並べてください
- `flow`: 選んだ記事をどういう意図で、どんな順序・繋ぎで紹介するか、自由文で記述してください

## 注意事項

- コーナーの趣旨（`content`）と目標放送時間（`target_duration_sec`）に合った記事を選んでください
- 候補記事がない場合でも最低1件は選ぶようにしてください（候補にある URL のみを使用してください）
- `selected_urls` に含める URL は必ず候補記事の中から選んでください
- `flow` は台本を書く際の指針となるよう、具体的な構成・繋ぎ方を記述してください
- 日本語で回答してください
