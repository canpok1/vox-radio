## vox-radio

AI を使ったポッドキャスト制作ツール

### Synopsis

vox-radio は AI を活用したポッドキャストエピソード制作 CLI ツールです。

記事収集・LLM による台本生成・音声合成・音声組み立て・コンテンツマニフェスト出力まで、
フルパイプラインをカバーします。

### Options

```
  -h, --help   help for vox-radio
```
