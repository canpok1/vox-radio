## vox-radio profile

プロファイルファイルを操作するサブコマンド群

### Synopsis

番組プロファイル YAML に関連するサブコマンドを提供します。

現在利用可能なサブコマンド:
  check  プロファイルファイルの内容を検証します

### Options

```
  -h, --help   help for profile
```
