03 — FEATURES

収集から配信まで、ひと続きで。

01

生成AIで原稿づくり

記事を選別し台本まで自動生成（多段 LLM パイプライン）。OpenAI 互換 API・Dify 対応。

02

VOICEVOX で音声合成

キャラクターごとに声色・口調を設定。感情に応じてスタイルを切替。

03

アセット演出

BGM・効果音・ジングルを合成。BGM のダッキングなど細かい演出も。

04

そのまま配信

RSS フィード生成で Podcast 購読、Slack 投稿にも対応。

05

エージェント連携

「ラジオ番組を作って」と頼めば設定編集まで任せられるスキルを同梱。

06

定期自動化

GitHub Actions のスケジュール実行で生成〜投稿を無人運用。

05 — USE CASES

使い道いろいろ

A

毎朝のニュース・天気番組

RSS をコーナーのデータソースにし、GitHub Actions の定期実行で生成〜配信を無人運用。

B

身内・コミュニティ向け

生成した番組を Slack に自動投稿。チーム内共有や趣味の発信に。

C

リスナー参加型番組

Google フォームのお便りを RSS 経由で取り込み、お便りコーナーとして番組に反映。

06 — INSTALL

導入はかんたん

Homebrew
macOS / Linux 向け。ffmpeg も同時に導入。
install スクリプト

バイナリDL
