# [D] コーナー要約プロンプト

以下のラジオ番組コーナーのセリフ一覧を読み、このコーナーの内容を要約してください。

## 入力

コーナー名: {{corner_title}}

```json
{{script_lines}}
```

## 出力形式

以下のJSON形式で回答してください。

```json
{
  "summary": "コーナー内容の要約（50〜150文字程度）",
  "points": ["要点1", "要点2", "要点3"]
}
```

## 注意事項

- 日本語で回答してください
- このコーナーで「何を話したか」を具体的に記述してください
- 要点は3〜5項目程度にしてください
- セリフが無い、または極端に少ない場合は `summary` を空文字、`points` を空配列にしてください
- 事実に基づいた内容のみ含めてください
