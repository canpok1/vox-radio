# [0] 記事要約プロンプト

以下の記事本文を読み、要点を3〜5行にまとめてください。

## 入力

```json
{{article}}
```

## 出力形式

以下のJSON形式で回答してください。

```json
{
  "summary": "記事全体の一行要約",
  "points": [
    "要点1",
    "要点2",
    "要点3"
  ]
}
```

## 注意事項

- 日本語で回答してください
- 要点は3〜5個にまとめてください
- 技術的な用語はそのまま使用してください
- 読者が記事を読まなくても概要がわかるように書いてください
- 事実に基づいた内容のみ含めてください
