# [C] 発音校正プロンプト（かな化取りこぼし・誤読の検出・修正）

以下のセリフ一覧について、`converted_text`（VOICEVOX 読み変換後テキスト）が `original_text`（元の台本テキスト）の意図した読みを正しく表しているか検証し、問題のある行のみ修正してください。

## 入力セリフ一覧

```json
{{lines}}
```

各エントリのフィールド:
- `corner_index`: コーナーの0始まりインデックス
- `line_index`: コーナー内のセリフの0始まりインデックス
- `original_text`: 元の台本テキスト
- `converted_text`: `direct` ステップが生成した読み変換テキスト

## 検出対象

以下の3種類の問題を検出してください。

1. **未変換**: VOICEVOX が誤読しやすい漢字が `converted_text` に残っている
   - 連濁（例: 頭突き→「ズツキ」。「あたまつき」は連濁が取りこぼされている）
   - 熟字訓（例: 今日→「キョウ」。「こんにち」は誤り）
   - 複合語の特殊読み（例: 小豆→「アズキ」。「しょうず」は誤り）
   - 音訓混在（例: 入場口→「ニュウジョウグチ」。「はいりばぐち」は誤り）
   - **かな化する漢字（自立語）の読みはカタカナで補正する** — 助詞の wa/e 化（下記）はひらがなのまま。読みが明確な漢字は漢字のまま残してよい
   - 助詞のwa/e化取りこぼし（例: 「私は」の助詞「は」が未置換 → 正: 「私わ」。「学校へ」の助詞「へ」が未置換 → 正: 「学校え」。連語末尾も同様: では→でわ、には→にわ、とは→とわ）
     - **語の一部の「は」「へ」（話す・部屋 等）は対象外** — 誤修正（false positive）を防ぐため、語中・語頭の「は」「へ」を助詞として置換しない（かな化が必要ならカタカナで: 話す→ハナス、部屋→ヘヤ）

2. **誤読**: かな表記が原文の意味と食い違う
   - 同音異義語の取り違え（例: 「行く」を「おこなく」と読むなど）
   - アクセント型ではなく読み自体が間違っている場合

3. **意味不明**: 日本語として意味が通らない変換

## 出力形式

修正が必要な行のみ `corrections` に含めてください。問題のない行は含めないでください。

```json
{
  "corrections": [
    {
      "corner_index": 0,
      "line_index": 2,
      "text": "ズツキするのだ",
      "reason": "頭突き→ズツキ（連濁の取りこぼし）"
    }
  ]
}
```

## 制約

- **セリフの内容・意味・件数・話者・順序は変えない** — 表記（読み）のみ修正する
- **問題のない行は `corrections` に含めない** — 修正不要な行を含めると正常な変換が上書きされる
- **修正テキストは行全体を設定する** — 元の `converted_text` の一部だけでなく、行全体の修正後テキストを `text` に設定する。表記規則は `direct` と同じ（かな化する自立語の読みはカタカナ、助詞の wa/e 化はひらがな、読みが明確な漢字は漢字のまま）
- **アクセント（ピッチ）のみの問題は対象外** — 読み（かな）自体が正しければ修正しない（VOICEVOX のアクセント処理に委ねる）

## 注意事項

- `corner_index` / `line_index` は入力の値をそのまま使用してください
- 問題がない場合は `corrections` を空配列にしてください
