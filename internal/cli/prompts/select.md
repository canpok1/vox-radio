# [A] 記事選別プロンプト

以下のコーナー情報と候補記事を元に、番組で実際に扱う記事を選別し、その選別理由を説明してください。
話の流れ（flow）の最終設計は別ステップで行うため、ここでは行いません。

## 出演キャスト

この回の出演キャスト一覧です（`appearance_count` は過去の出演エピソード数。0 = 今回が初登場）。
`last_episode_number` は前回出演した回番号（省略時は過去に出演なし）。別途渡る放送回との差で「何回ぶりの出演か」を判断できます。

```json
{{casts}}
```

## コーナー情報

`appearance_count` は今回を含むこのコーナーの扱い回数（`1` = 今回が初めて = 新コーナー）。
`last_episode_number` は前回このコーナーを扱った回番号（省略時は過去に扱いなし）。別途渡る放送回との差で「何回ぶりか」を判断できます。

```json
{{corner}}
```

## 候補記事（タイトルと URL のみ）

```json
{{articles}}
```

## 出力形式

以下のJSON形式で回答してください。

```json
{
  "selected_urls": [
    "https://example.com/article/1",
    "https://example.com/article/2"
  ],
  "selection_reason": "AIチップ性能向上の記事が最も注目度が高く、次にその普及課題の記事を選んだ。この2本でコーナーの趣旨を網羅できる"
}
```

- `selected_urls`: 番組で紹介する記事の URL を**紹介順**に並べてください
- `selection_reason`: なぜこの記事をこの順で選んだか・どんな意図か、自由文で記述してください

## 注意事項

- コーナーの趣旨（`content`）と目標放送時間（`target_duration_seconds`）に合った記事を選んでください
- 候補記事がない場合でも最低1件は選ぶようにしてください（候補にある URL のみを使用してください）
- `selected_urls` に含める URL は必ず候補記事の中から選んでください
- `selection_reason` は記事の選び方の意図・根拠を記述してください（話の流れの詳細設計は不要）
- 日本語で回答してください
