# 作業メモ運用ルール

## ステップ完了時のチェックボックス更新

各ステップを完了したら、**その場で**作業メモのチェックボックスを `[x]` に更新すること。

- PR 作成・マージ・コミットなど節目となる操作が完了したら即時更新する（セッション終了時のまとめ更新は禁止）

## セッション引き継ぎ時の実状態確認

セッション引き継ぎ時は、作業メモのチェックボックスを鵜呑みにせず、以下のコマンドで git の実態（正）を確認してから作業を再開すること。

```bash
git log --oneline -5   # 直近コミット（完了済みステップ）を確認
git status             # 未コミット変更の有無を確認
```
