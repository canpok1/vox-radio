# 作業メモ運用ルール

## ステップ完了時のチェックボックス更新

各ステップを完了したら、**その場で**作業メモのチェックボックスを `[x]` に更新すること。

- PR 作成・マージ・コミットなど節目となる操作の直後も忘れずに更新する
- セッション終了時にまとめて更新するのではなく、完了のたびに即時更新する

## セッション引き継ぎ時の実状態確認

セッション引き継ぎ時は、作業メモのチェックボックスを鵜呑みにせず、以下のコマンドで実際の完了状態を確認してから作業を再開すること。

```bash
git log --oneline -10
git status
```

- チェックボックスが未更新のまま引き継がれる場合があるため、git の実態を正とする
