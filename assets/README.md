# assets ディレクトリ

BGM・効果音（SE）・ジングル・カバー画像を配置するディレクトリです。

## ディレクトリ構成

```
assets/
├── bgm/               # BGM音源
│   └── talk.mp3       # トーク中に流すBGM（ダッキング対応・ループ再生）
├── se/                # 効果音（Sound Effect）
│   ├── chime.wav      # チャイム（コーナー開始等）
│   └── transition.wav # トランジション音（話題転換等）
├── jingle/            # ジングル
│   ├── opening.mp3    # オープニングジングル
│   └── ending.mp3     # エンディングジングル
└── cover.jpg          # Podcastのカバー画像（3000×3000px推奨）
```

各ファイルパスと音量設定は `config/assets.yaml` で管理しています。

## ライセンスに関する注意

Podcastは原則公開配信のため、BGM・SE・ジングルは**ライセンス確認が必須**です。

### リポジトリへの同梱について

- ライセンスによってはリポジトリへの同梱（Gitでの管理）が禁止されている場合があります
- 本リポジトリに含まれている音声ファイルは開発・テスト用のダミー素材（無音ファイル）です
- 本番運用時は以下の手順に従って実際の音源に差し替えてください

## 音源の調達手順

### フリー音源の調達先候補

| サイト | 特徴 |
|--------|------|
| [DOVA-SYNDROME](https://dova-s.jp/) | BGM・SE多数。商用利用可。クレジット表記が必要な場合あり |
| [甘茶の音楽工房](https://amachamusic.chagasi.com/) | BGM中心。商用利用可。クレジット表記必要 |
| [効果音ラボ](https://soundeffect-lab.info/) | SE中心。商用・同梱可のものが多い |
| [魔王魂](https://maoudamashii.jikkokuhyo.jp/) | BGM・SE多数。クレジット表記必要 |

### 音源選定時の確認事項

音源を選定する際は、以下の点を規約で必ず確認してください。

- [ ] 商用利用の可否
- [ ] クレジット表記の要否
- [ ] リポジトリへの同梱（Gitでの管理）の可否
- [ ] 配布・再配布の可否

### 音源の差し替え手順

1. 上記の確認事項をクリアした音源を用意する
2. 以下のファイルを置き換える：
   - `assets/bgm/talk.mp3` — トーク用BGM（ループ再生されます）
   - `assets/se/chime.wav` — チャイム効果音
   - `assets/se/transition.wav` — トランジション効果音
   - `assets/jingle/opening.mp3` — オープニングジングル
   - `assets/jingle/ending.mp3` — エンディングジングル
3. `config/assets.yaml` の音量・フェード設定を番組に合わせて調整する
4. `assets/cover.jpg` を番組に合ったカバー画像（3000×3000px推奨）に差し替える

### カバー画像について

`cover.jpg` はPodcastのカバーアート（チャンネルアート）です。  
Apple PodcastsなどのPodcastアプリに表示されます。

- 推奨サイズ: 3000×3000px（正方形）
- 最小サイズ: 1400×1400px
- フォーマット: JPEG または PNG
- ファイルサイズ: 512KB 以下推奨

### クレジット表記が必要な場合

`config/podcast.yaml` の `description` 末尾にクレジットを追記してください。

```yaml
description: |
  毎日5分のニュースラジオ

  【使用音源】
  BGM: 〇〇（〇〇様）- https://example.com
  SE: 〇〇 - 効果音ラボ
```

## テスト用ダミー素材について

このリポジトリに含まれる音声ファイルはすべて**開発・テスト用の無音ファイル**です。  
実際の音声は含まれていません。本番運用時は必ず実際の音源に差し替えてください。
